Название продукта:
Инструментальные средства объектной разработки программных систем S_Технология (SDK S_Технология).
Object oriented tools for software system development S_Technology (SDK S_Technology).

Состав инструментальных средств приведен в документе
"Состав SDK S_Технология.pdf".
Состав документации по SDK S_Технология приведен в документе
"Состав документации SDK S_Технология.pdf".

Внимание!
Для правильной работы SDK S_Технология необходимо настроить базу данных и создать дополнительные объекты с помощью SQL-скриптов.
Инструкция по установке и настройке инструментальных средств для различных СУБД приведена в документе
"IM Установка SDK S_Технология.pdf".

Ограничения демонстрационной версии:
При редактировании выборки дата-класса может быть изменено не более 10 строк выборки. 
Под изменением строки понимается операция добавления/удаления/исправления.
